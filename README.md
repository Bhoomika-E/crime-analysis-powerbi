# Crime Data Analysis using Python & Power BI 

##  Project Overview
A comprehensive analysis of crime patterns in Cambridge to identify hotspots and seasonal trends. This project was completed as part of my 5th-semester BCA curriculum.

##  Tech Stack
- **Data Cleaning:** Python (Pandas, NumPy)
- **Visualization:** Power BI, Matplotlib
- **Data Source:** Kaggle

##  Key Insights
- Identified a 15% increase in specific crimes during winter months.
- Mapped 5 major 'hotspot' zones using Power BI Map visuals.
